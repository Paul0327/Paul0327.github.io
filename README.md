Headings

Use # symbols to create headings. The number of # determines the heading level:

# Heading 1
## Heading 2
### Heading 3
Copiar
Styling Text

Bold: **text** or __text__

Italic: *text* or _text_

~~Strikethrough~~: ~~text~~

Bold and Italic: ***text***

Lists

Unordered Lists: Use -, *, or +:

- Item 1
- Item 2
Copiar
Ordered Lists: Use numbers followed by periods:

1. First
2. Second
Copiar
Task Lists: Use - [ ] for incomplete and - [x] for completed tasks:

- [x] Task 1
- [ ] Task 2
Copiar
Links and Images

Links: [Link Text](URL)

Images: ![Alt Text](Image URL)

Code and Syntax Highlighting

Inline Code: Use backticks: `code`

Code Blocks: Use triple backticks with optional language specification:

def hello_world():
print("Hello, World!")
Copiar
Blockquotes

Use > to create blockquotes:

> This is a blockquote
Copiar
Tables

Create tables using pipes (|) and hyphens (-):

| Header 1 | Header 2 |
|----------|----------|
| Row 1 | Data 1 |
| Row 2 | Data 2 |
Copiar
Mentions and References

Mention users or teams: @username

Reference issues or pull requests: #123

Emojis

Add emojis using :emoji_name: syntax:

:smile: :rocket:
Copiar
Footnotes

Add footnotes using [^1] and define them at the bottom:

Here is a footnote reference[^1]

[^1]: This is the footnote text
Copiar
Alerts

Use blockquote syntax with [!NOTE], [!TIP], [!IMPORTANT], [!WARNING], or [!CAUTION] for alerts:

> [!NOTE]
> This is a note
Copiar
Advanced Features

Diagrams: Use mermaid syntax for flowcharts and diagrams.

Mathematical Expressions: Use LaTeX syntax for math equations:

$E = mc^2$
